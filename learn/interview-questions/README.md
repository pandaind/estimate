# Interview Questions

High-value interview questions based on every technology and concept used in the EstiMate project.  
Use these to prepare for technical interviews or to test your understanding after working through the learning guides.

## Files

| File | Topics covered |
|------|----------------|
| [frontend.md](frontend.md) | React, Vite, React Router, Axios/REST, WebSocket/STOMP, Tailwind CSS, Framer Motion, Vitest + Testing Library, Accessibility |
| [backend.md](backend.md) | Spring Boot, REST Controllers, Service Layer / DI, JPA + Hibernate, Spring Security + JWT, Spring WebSocket, Exception Handling, Flyway, JUnit 5 + Mockito, Pagination, Concurrency |
| [system-design.md](system-design.md) | Real-time collaboration architecture, stateless auth, database design, Docker, horizontal scaling |

## How to use these

Each question is marked with a difficulty rating:

- 🟢 **Foundational** — expected for any junior / mid-level role
- 🟡 **Intermediate** — shows depth; expected for mid-level and above
- 🔴 **Advanced** — system-design or nuanced trade-off; expected for senior roles

A **sample answer** (or key points to hit) follows each question so you can self-check.
